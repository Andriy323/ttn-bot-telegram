# ttn-bot-telegram

/start - Початок роботи з ботом, вітальне повідомлення.

## Змінні для шаблону ТТН (template.html)

Нижче наведено перелік усіх змінних, які були додані до HTML-шаблону для автоматичної генерації ТТН:

- `{{ttn_number}}` - Номер ТТН
- `{{ttn_date}}` - Дата складання ТТН
- `{{car_info}}` - Автомобіль (марка, модель, тип, реєстраційний номер)
- `{{trailer_info}}` - Причіп/напівпричіп
- `{{driver_info}}` - Водій (ПІБ, посвідчення)
- `{{carrier_info}}` - Автомобільний перевізник
- `{{shipper_info}}` - Вантажовідправник
- `{{consignee_info}}` - Вантажоодержувач
- `{{loading_point}}` - Пункт навантаження
- `{{unloading_point}}` - Пункт розвантаження
- `{{weight_brutto}}` - Маса брутто (числом у загальному блоці та в таблицях)
- `{{driver_signature_name}}` - ПІБ водія/експедитора (для поля "отримав водій/експедитор")
- `{{tare_and_brutto}}` - Загальна вага/маса з вантажем та маса брутто (наприклад: `14,68/39,54`)
- `{{cargo_name}}` - Найменування вантажу
- `{{places_words}}` - Кількість місць (словами)
- `{{weight_brutto_words}}` - Маса брутто (словами)
- `{{shipper_manager}}` - Здав (відповідальна особа вантажовідправника)
- `{{weight_netto}}` - Маса нетто (значення в таблиці на другій сторінці)
- `{{packing_type}}` - Вид пакування (наприклад: `насипом`)
